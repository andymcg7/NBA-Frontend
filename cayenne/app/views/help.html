NBA League Score Entry Help



Welcome to NBA's new HTML5 league score entry system. This is part of a large project to remove any reliance on Flash Player from the Northumberland Leagues and Rankings.


There are some small differences between this new version and the old version, so please continue to read if you are having any issues.

The captain's email address is no longer filled in automatically - the email address should be entered each time a match is entered.

Players are now chosen from a dropdown list rather than typed, so if you cannot see the player name you need to complete the match card, please email the League Secretary with the details.

To save time when entering the match scores, you should only type in the losing score in each set. The other score will be automatically filled in.

As before, if a match was conceded, i.e. not played at all, tick the conceded box, AND enter 21-0 21-0 to the winning side. If a match was started and not finished (due to injury for example) DO NOT tick the conceded box, but type in the score as if the winning team won all of the points after the stoppage.


Please report any bugs or issues with the system to the webmaster.
